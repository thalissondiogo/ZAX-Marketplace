- Dark Mode
- Design Responsivo

## 💻 Como Rodar
1. Clone este repositório:
   ```bash
   git clone https://github.com/thalissonsenai-boop/ZAX-Marketplace.git
   ```
2. Abra o diretório do projeto:
   ```bash
   cd ZAX-Marketplace
   ```
3. Abra o `index.html` diretamente no seu navegador.
