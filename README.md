## ✨ Features

### 🛡️ Confiança & Segurança
- Trust bar animada com selos de garantia
- Badge de compra verificada nos reviews
- Selo de qualidade em todos os produtos
- Garantia de devolução em 7 dias

### 🌍 Internacionalização
- **3 idiomas:** Português, English, Español
- **3 moedas:** BRL (R$), USD ($), EUR (€)
- Conversão automática de preços

### 🔍 Busca Avançada
- Busca em tempo real com sugestões
- Filtros por categoria (Eletrônicos, Moda, Casa, Acessórios)
- Ordenação: Destaques, Menor/Maior preço, Avaliação, Mais vendidos

### ⭐ Avaliações & Reviews
- Sistema de estrelas (1-5)
- Modal com distribuição de notas
- Badge de "Compra Verificada"
- Contador de avaliações úteis

### 🏆 Programa de Fidelidade
- **ZAX Points** — ganhe pontos a cada compra
- 4 níveis: Bronze → Prata → Ouro → Diamante
- Descontos progressivos de 0% a 15%

### 🎨 Design Premium
- **Dark Mode** com toggle
- Glassmorphism e gradientes
- Micro-animações e transições suaves
- Design totalmente responsivo
- Acessibilidade WCAG (ARIA, focus, reduced motion)

### 🛒 Carrinho Inteligente
- Barra de progresso para frete grátis
- Controles de quantidade (+/−)
- Persistência via localStorage
- Checkout via WhatsApp

### 📱 Mobile-First
- Bottom navigation para mobile
- FAB do WhatsApp flutuante
- Cards adaptativos
- Touch-friendly

### ❤️ Extras
- Wishlist com persistência
- Quick View modal
- Social sharing (Web Share API)
- Newsletter com design gradient
- Carrossel de destaques

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/SEU_USUARIO/ZAX-Marketplace.git
```

2. Abra `index.html` no navegador:
```bash
# Ou use um servidor local
npx serve .
```

Não requer instalação de dependências — 100% HTML/CSS/JS puro.

## 📁 Estrutura

```
ZAX_Ecommerce_HTML/
├── index.html              # Página principal
├── css/
│   └── styles.css          # Design system completo
├── js/
│   ├── config.js           # Configurações centralizadas
│   ├── products-data.js    # Catálogo de produtos (24 itens)
│   └── script.js           # Lógica principal (i18n, carrinho, etc.)
└── assets/
    └── images/
        ├── favicon-32x32.png
        └── cart-icon.png
```
